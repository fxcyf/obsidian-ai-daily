# Feature: 个人信息 Feed 流

**状态**: ✅ Phase 1 已实现

## 概念

将插件从纯知识库对话扩展为个人信息聚合工具。用户定义关注主题后，AI 自动搜集相关信息，形成可浏览的 feed 流。

## 初步想法

- 用户在文档中记录关注的主题/关键词
- AI 定期（或按需）搜集相关信息
- 以 feed 流形式呈现，供用户后续浏览
- 与现有知识库功能结合，搜集到的信息可沉淀到 Wiki

## 待确定

- 信息来源：RSS？网页搜索？API？
- 更新频率：手动触发 vs 定时
- 展示形式：侧边栏 feed？生成 daily note？
- 与现有对话功能的交互方式
